О SashenNet

SashenNet — это независимая частная сеть с собственными сайтами на домене .sashen. Здесь нет алгоритмов, рекламы и слежки — только то, что создаём мы сами.

Что это такое?

SashenNet — это что-то среднее между личным интернетом и маленьким даркнетом для своих. Все сайты живут в одном репозитории и доступны через единый поисковик на главной странице. Вместо привычных .com или .ru — домены заканчиваются на .sashen.

Как добавить сайт?

На данный момент создание новых сайтов в сети SashenNet закрыто. Добавлять сайты может только разработчик — Nyxnotz. В будущем это может измениться.

Кто это сделал?

SashenNet сделан Nyxnotz. Это личный проект — эксперимент с идеей собственного интернета внутри одного GitHub репозитория.

Технологии

Чистый HTML, CSS и JavaScript. Никаких фреймворков, никакого сервера. Хостинг — GitHub Pages. Поиск работает через fetch из sites.json.
